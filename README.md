# FirstLedger
A simple ledger API built using .NET 8.0 with features like:
- Money movements (deposits and withdrawals)
- Current balance viewing
- Transaction history

## Getting Started

### Prerequisites
- .NET 8.0 SDK
- Visual Studio or VS Code

### Installation
```bash
git clone https://github.com/j-hashemi/FirstLedger.git
cd FirstLedger
dotnet run
